🚧 Road Accident Dashboard (Excel)

An interactive Excel dashboard analyzing road accident casualty data using PivotTables, PivotCharts, and slicers. The dashboard provides a visual summary of casualty severity, vehicle types, weather conditions, road surface, lighting, and location to help identify accident patterns and risk factors.
Overview

Total Casualties: 417,883

The dashboard is built entirely in Excel using pivot tables/charts connected through slicers, allowing users to filter the entire report dynamically by Weather Conditions and Urban/Rural location.
